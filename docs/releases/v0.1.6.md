# Clipin v0.1.6

`v0.1.6` 基于当前已提交的 `HEAD` 发布，不包含任何未提交改动。相较 `v0.1.5`，这次更新把启动引导、更新提醒、固定项浏览策略、搜索链路和本地化体验收成了一次真正可交付的产品迭代。

## Highlights

- 首次启动引导与权限路径完整上线：新增 welcome scene、权限步骤和 release 可复现入口，并修复方向键、按钮反馈、授权后自动重启等关键回归。
- 固定项浏览策略可配置：普通列表中的 pinned 现在支持混排、单独分组或仅在 pinned 视图显示，launcher 默认入口也可选择全部、固定项或记住上次视图。
- 搜索体验重做：支持拼音搜索、IME 未确认实时搜索、按真实使用频率排序，并修复 pinned 条目被错误隐藏、可见页分页错误和若干排序回归。
- 应用内更新提醒上线：关于页可检查 GitHub Releases、查看更新说明并直达下载页。
- 中英文文案补齐：菜单标题、动态提示、预览计数、快捷键名、文件摘要和新增设置项统一走完整本地化链路，不再出现半英半中。

## Fixes

- 修复 `Tab` 浏览模式在首尾不循环的问题。
- 修复“仅在 pinned 视图显示”时第一页可能被隐藏条目吃满，导致列表误显示为空的问题。
- 修复同内容去重时 `paste_count` 被重置，热门条目排序失真的问题。
- 修复 `zhu yi` / `zhu'yi` 这类拼音 preedit 无法稳定命中中文条目的问题。
- 修复搜索态误继承 pinned 浏览语义、导致明明存在的条目搜不到的问题。
