# Clipin v0.1.3

`v0.1.3` 基于当前已提交的 `HEAD` 发布，不包含任何线程中的未提交改动。相较 `v0.1.2`，这次更新已经从“补丁修复”走到了明显的产品迭代。

## Highlights

- 全新设置中心：重做设置窗口结构，加入侧边栏导航、语言切换、外观模式和视觉主题选项，并补齐键盘导航、首次点击响应和动效细节。
- 更细化的隐私采集策略：继续强制跳过密码管理器等 `ConcealedType` 内容，同时新增对 `TransientType` / `AutoGeneratedType` 的可配置跳过策略，默认更保守。
- 自动备份上线：支持备份到 iCloud Drive 或任意文件夹，可按剪贴板变化、日、周、月执行，并持久化最近一次备份时间。
- 中英文界面：新增简体中文 / English 本地化，同时修复中文搜索与多语言切换中的回归问题。
- 预览与视觉升级：支持文本链接点击、图片文件预览，补上 `ClipinTheme` 设计系统，并把预览区迁移到更贴近 macOS 的 glass 材质风格。

## Fixes

- 修复设置窗口使用浮动层级导致 `Cmd+W` 误发给其他应用的问题。
- 修复设置窗口可见时错误还原上一个应用焦点的问题。
- 修复 `ShortcutRecorder` 非录制状态拦截系统快捷键的问题。
- 修复 notice 会随滚动消失、展示时间偏短的问题。
- 修复 `keyBadge` 类型不兼容导致的编译失败。
- 汇总吸收多轮 review 中发现的问题，覆盖设置页交互、文案退化和若干 UI 细节。
